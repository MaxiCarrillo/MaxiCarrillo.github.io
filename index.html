BIRABAR

  • Angular
  • MongoDB
  • NodeJS

Aplicación web creada para un bar de la provincia de Jujuy con el fin de que estos puedan gestionar y ver los informes de su negocio.

Este proyecto fue realizado en grupo, ayudé muchisimo para que la pagina sea responsive y además mi principal tarea fue manejar la logica de las ofertas, por lo cual tuve que estar metido tanto en el Back-end (Express, JWT, Cors) como Front-end (Bootstrap, Toastify, Typescript).

PUPILIFE

  • Spring Framework
  • MySQL
  • Java

Pagína web realizada en conjunto con mi novia con el proposito de brindar consejos y cuidados para salud.

El proyecto utiliza el patron MVC (Model View Controller) y relaciones JPA para la lógica del negocio. Se utilizo Thymeleaf para trabajar acople con el patron MVC y Bootstrap para la realizacion del Front-end.
